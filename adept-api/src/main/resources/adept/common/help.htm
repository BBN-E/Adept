Web Service Output

 

The web service output consists of a table of contents which points to a set of tables.� Each kind of annotation supported by the selected algorithm will appear in its own table.� The first table is the input document itself, and the last table is the list of tokens found in the document.� The tables are documented in order below.

 

Document

Row

Definition

Document Value

The original text of the document

Type:

The original document format

URI:

The location of the document

Language:

The language of the document.� At this time only English is supported.

Headline:

The document title if any

 

Entity Mentions and Coreferences

Column

Definition

ResolvedEntityId

The index of the entity being mentioned

SequenceId

The sequential index within the document

Entity Mention Value

The text of the entity mention

Begin

The index of the first token

End

The index of the last token

MentionType

The type of the mention, if any

EntityType

The type of the entity, if any

Context

The context where this appears within the document

 

Relations

Column

Definition

RelationId

The sequential index within the document

Relation Type

 

Begin

The index of the first token in each argument

End

The index of the last token in each argument

Argument Value

The text of each argument

Argument Type

The type of each argument

Confidence

A value from 0 meaning uncertain to 1 meaning certain

Context

The context where this appears within the document

 

Sarcasms

Column

Definition

SarcasmId

The sequential index within the document

Sarcasm value

The text of the sarcasm

Begin

The index of the first token

End

The index of the last token

Judgment

Positive for sarcasm or negative for non-sarcasm

Confidence

A value from 0 meaning uncertain to 1 meaning certain

 

Committed Beliefs

Column

Definition

SequenceId

The sequential index within the document

Committed Belief Value

The text of the committed belief

Begin

The index of the first token

End

The index of the last token

Modality

The kind of belief, intention or other commitment being expressed

 

Paraphrases

Column

Definition

Paraphrase Value

The text of the paraphrase

POS Tag

The part of speech if any

Triples

Column

Definition

Entity

The token index, starting from zero

Slot

The text of the token

Value

The first character position in the token, and the first character position after the token.

 

Tokens

Column

Definition

The token index, starting from zero

The text of the token
